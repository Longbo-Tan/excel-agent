# Excel Agent

AI Agent for Excel — 侧边栏聊天窗口，用自然语言操作 Excel。

## 🚀 一键安装

### Windows
```
下载 install.bat → 双击运行 → 按提示操作
```

### Mac
```
下载 install.sh → 终端运行 → 按提示操作
```

## 安装步骤

脚本会自动复制安装 URL 到剪贴板，然后：

1. 打开 Excel
2. **插入** → **我的加载项** → **从文件安装**
3. Ctrl+V / Cmd+V 粘贴 URL
4. 点击确定
5. 在侧边栏设置 API Key

## 功能

- 🤖 ReAct Agent 多步执行
- 📊 读/写/格式/公式/图表
- ⚙ Skills + 自定义规则
- 💬 多对话管理
